Theory & Algorithm

The SWAP test is a canonical quantum subroutine that estimates the overlap $|\langle\psi|\phi\rangle|^2$ between two quantum states using a single ancilla qubit and $n$ controlled-SWAP (Fredkin) gates. We encode each image as a normalised amplitude vector and run the SWAP test to obtain a quantum similarity score.

Step 1 — Amplitude Encoding

Flatten the image pixel values into a vector, normalise to unit norm, and encode each component as a probability amplitude:

$$\mathbf{v} \;=\; \bigl[f(0,0),\; f(0,1),\; \ldots,\; f(H-1,W-1)\bigr] \;\in\; \mathbb{R}^{H \cdot W} \tag{1}$$
$$\hat{\mathbf{v}} \;=\; \frac{\mathbf{v}}{\|\mathbf{v}\|_2}, \qquad |\psi_A\rangle \;=\; \sum_{i=0}^{2^n - 1} \hat{v}_i\,|i\rangle \tag{2}$$

where $n = \lceil\log_2(H \cdot W)\rceil$ qubits are required per image.

Step 2 — SWAP Test Protocol

A single ancilla qubit mediates the comparison of two $n$-qubit states $|\psi_A\rangle$ and $|\psi_B\rangle$. The circuit proceeds in three steps:

$$\text{(i)}\;\; H\,|0\rangle_\text{anc} \;\longrightarrow\; \tfrac{1}{\sqrt{2}}\bigl(|0\rangle + |1\rangle\bigr) \tag{3}$$
$$\text{(ii)}\;\; \text{Apply } \text{CSWAP}\!\left(\text{anc},\,|\psi_A\rangle_i,\,|\psi_B\rangle_i\right), \quad i = 0,\ldots,n-1 \tag{4}$$
$$\text{(iii)}\;\; H \;\text{ on ancilla, then measure:}\quad P(\text{anc} = 0) \;=\; \frac{1}{2} + \frac{1}{2}\,|\langle\psi_A|\psi_B\rangle|^2 \tag{5}$$

The similarity score is then recovered as:

$$\text{Quantum Similarity} \;=\; 2\,P(0) - 1 \;\in\; [0,\,1] \tag{6}$$

A score of 1 means the two states are identical; a score near 0 means they are approximately orthogonal (maximally different).

Circuit Diagram

Qiskit SWAP test circuit: one ancilla qubit (anc) flanked by two Hadamard gates; amplitude-encoding blocks for images A and B; three CSWAP (Fredkin) gates coupling corresponding qubit pairs. The measurement on the ancilla yields $P(0)$, from which the quantum similarity score is computed.

Interpreting the Score

Score $= 1.0$  →  identical images $\bigl(|\langle\psi|\psi\rangle|^2 = 1\bigr)$

Score $\to 0$  →  orthogonal / maximally different

Score $= 0.5$  →  statistically uncorrelated

Classical Baselines Computed

Cosine similarity: $\;\hat{\mathbf{v}}_A \cdot \hat{\mathbf{v}}_B$

MSE: $\;\tfrac{1}{N}\sum_i(A_i - B_i)^2$

Histogram intersection: $\;\sum_k \min(h_A^k, h_B^k)$

03

Implementation

The matching module (src/matching.py) handles amplitude encoding, SWAP test circuit construction, and shot-based estimation internally. It also exposes a batch() helper for ranking a gallery of images by similarity to a query. 

import matching
import numpy as np

# Compare two arbitrary images
img_a = np.random.rand(16, 16) * 255
img_b = np.random.rand(16, 16) * 255

result = matching.run(img_a, img_b, shots=8192)

print(f"Quantum similarity:     {result['quantum_similarity']:.4f}")
print(f"Classical cosine:       {result['classical_cosine']:.4f}")
print(f"Histogram intersection: {result['histogram_intersection']:.4f}")

# Rank a gallery of images by similarity to the query
gallery = [img_a, img_b, np.zeros((16,16)), np.ones((16,16))*128]
ranked  = matching.batch(img_a, gallery, shots=4096)

for r in ranked:
    print(f"  rank {r['index']}: score = {r['quantum_similarity']:.4f}")
